Data Information

Data Sources

Public Health Wales Rapid COVID-19 Surveillance is an interactive data dashboard which allows the public and media in Wales to find out more about the virus in Wales.
Data is updated daily at 12 noon. The data used in this portal is accumulated over time from Public Health Wales's downloadable data available on the dashboard. The data is subsequently processed to calculate differences in the data and cumulative data.

Stats Wales is a free to use service that allows you to view, manipulate, create and download Welsh data.
The data used in this portal is accumulated over time. The data is subsequently processed to calculate differences in the data and cumulative data.

GOV.UK Coronavirus (COVID-19) in the UK is the official UK Government website for data and insights on COVID-19.
The data used in this portal is accumulated over time using the available API. The data is subsequently processed to calculate differences in the data and cumulative data.
